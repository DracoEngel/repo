## Open Graph (previsualización al compartir)

El sitio incluye metatags `og:*` y `twitter:*` para que al compartir el link se vea una tarjeta con imagen.

- Imagen actual (placeholder): `assets/img/og-image.svg` (1200×630)
- Recomendación para producción: exportar a `PNG/JPG` y cambiar `og:image` a esa ruta, porque algunas plataformas no renderizan SVG como preview.
